# Contribuer

On n'a besoin d'aide pour mener à bien cette révolution citoyenne.
Rejoignez-nous!

- [Groupe Facebook des volontaires](https://www.facebook.com/groups/listescitoyennes.volontaires)

## Comment ajouter des infos à propos d'une liste?

Le plus rapide c'est de directement modifier le fichier `lists.json` et de faire une "pull request": [Github](https://github.com/xdamman/pourquellelistevoter/blob/master/data/lists.json).

Si trop compliqué, vous pouvez aussi simplement éditer le [google spreadsheet](https://docs.google.com/spreadsheets/d/138MhE96bxlQ54bbH_W4NOZSo8jjcdyzPyIpwuHJQoRs/edit?usp=sharing)
 (en espérant que quelqu'un se porte volontaire pour faire une pull request avec les infos issues de ce tableau).
